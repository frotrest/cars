
<h1 align="left">🚗 Cars</h1>

###

<p align="left">Современный одностраничный сайт, предназначенный для презентации автомобильных услуг, моделей или бренда. Подходит для автосалонов, проката авто, тюнинг-студий и других авто-проектов.</p>

###

<h2 align="left">🧑‍💻 Технологии сайта</h2>

###

• 	Vite — мгновенная сборка и dev-сервер

• 	HTML5 + CSS3 — адаптивная и семантическая верстка

• 	JavaScript — интерактивные элементы

• 	GitHub Pages — хостинг лендинга

###

<h2 align="left">📁 Структура проекта</h2>

###

```
cars/
```
